# adv_survival

마인크래프트 도전과제 서바이벌 플러그인입니다.

## 모드 요약

- 도전과제를 클리어할 때마다 살아있을 수 있는 시간이 추가됩니다.
- 플레이어가 죽을 때마다 남은 시간이 일정량 감소합니다.
- 시간이 모두 떨어지면 게임이 종료됩니다.
- 목표는 최대한 오래 살아남는 것입니다.

## 플레이 명령어

- `/advsurvival start` : 도전과제 서바이벌 시작
- `/advsurvival stop` : 현재 게임 종료
- `/advsurvival time` : 현재 남은 시간 확인
- 사용 가능한 별칭: `/asurvival`, `/advs`
